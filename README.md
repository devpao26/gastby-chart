# React Gastby Chart

# A chart thats is built on top of react gatsby, node js and chartting library with local json data served.

Navigate into the project and start it up.

cd gastby-chart/
npm install
npm start

Project is now running at http://localhost:8000.

Note: You'll also see a second link: http://localhost:8000/___graphql. This is a tool you can use to experiment with querying your data.
